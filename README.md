# Alex — AI Financial Research Platform

A production-grade multi-agent AI system built on AWS.

## Architecture
- **Researcher Agent** — App Runner + Bedrock + Playwright MCP
- **RAG Pipeline** — SageMaker embeddings + S3 Vectors
- **Multi-agent** — SQS orchestrated Lambda agents
- **Frontend** — Next.js + Clerk + CloudFront
